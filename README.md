# LogisticsHub

LogisticsHub is a microservices-based backend project built in a production-style way.

## Services

- Gateway
- InventoryService
- ShipmentService

## Local setup

| Service | URL |
|---|---|
| Gateway | http://localhost:5100 |
| InventoryService | http://localhost:5101 |
| ShipmentService | http://localhost:5102 |

## Build

```powershell
dotnet build .\LogisticsHub.sln
